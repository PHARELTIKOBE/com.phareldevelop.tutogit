##Création d'une nouvelle branche

```bash
git checkout -b NOM_BRANCHE
```

Pour les bonnes pratiques , on va intégrer la notion de revue de code .Pour cela
on va  créer une branche , faire des modifications , les envoyers sur le dépêt distant , puis
créer une pull request pour demander une revue de code
